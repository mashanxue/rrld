登录注册

  • 注册时，进行手机验证，收不到短信怎么办？

    1. 请确认手机是否安装短信拦截或过滤软件；

    2. 请确认手机是否能够正常接收短信（信号问题、欠费、停机等）；

    3. 短信收发过程中可能会存在延迟，请耐心等待，短信在30分钟内均有效；

    4. 您还可以联系客服，寻求帮助（95770）。

  • 注册成功之后昵称可以修改吗？

    不可以。昵称一旦注册成功不能再修改。

  • 在renrendai.com上注册后，可以不开立银行存管户就进行出借吗？

    不可以。存管子账户用于分账管理用户的交易与资金信息，如不开立，无法进行后续的操作。

  • 在验证民生账户/重置密码时，手机收不到验证码怎么办？

    1. 请确认手机是否安装短信拦截或过滤软件；

    2. 用户打电话要求屏蔽民生95568的短信，致使其收不到。需用户电话95568开启短信提醒服务；

    3. 部分运营商短信存在到达率问题；

    4. 目前如果60秒收不到短信，可以点击“语音验证码”，民生系统会通过拨打客户手机号进行语音通知。

账户密码

  • 如何修改登录密码？

    打开【我的账户】--【账户设置】--【账户信息】页面，找到【登录密码】信息栏，点击【修改】，修改登录密码。

  • 如何找回登录密码？

    打开登录页面，点击【忘记密码】；跳转至【找回密码】页面，可以选择通过绑定的邮箱或者绑定的手机号找回登录密码。

  • 通过邮箱找回登录密码时，收不到邮件，怎么办？

    请您在垃圾邮件中查找，如果没有，则可能是您邮箱服务器存在问题，建议您通过绑定的手机号找回登录密码 。

  • 实现资金存管后，如何设置交易密码？

    在您开通民生银行存管账户时，您需要设置民生银行存管账户交易密码，此密码将做为用户充值、提现、购买债权标的、加入自动投标工具时的交易密码。若您需要修改或找回民生银行存管账户交易密码，需要进入民生银行存管账户管理页面进行操作。

  • 如何修改交易密码？

    打开【我的账户】--【账户设置】--【账户信息】页面，找到【进入存管账户】信息栏，点击【密码重置】，修改交易密码。

  • 民生银行存管账户交易密码和民生银行卡交易密码有关系吗？

    没有关系。您通过renrendai.com平台进入民生银行存管系统设置的民生银行存管账户交易密码为您在renrendai.com平台进行出借时的资金交易密码，用于您在使用民生银行存管子账户内资金时的鉴权，而您的民生银行卡，是您独立至民生银行开立的实体银行账户，与您在平台的存管子账户并非同一账户。

  • 我的手机号出现变更，该如何修改？

    用户可以通过平台自行更改。打开【我的账户】--【账户设置】--【账户信息】页面，找到【绑定手机】，点击修改，修改绑定手机号码。

开户

  • 如何开立民生银行存管账户？

    1、用户注册人人贷账户成功后，会引导用户前往民生银行页面开通存管账户。在开户页面输入【姓名】、【身份证号】、【银行卡号前6位】，确认无误后，点击【立即开通】按钮。

    2、跳转到民生银行开户页面，输入完整的【银行卡号】设置【民生交易密码】、验证【银行预留手机号】完成后，开户成功。若您的【人人贷注册手机号】与【银行预留手机号】不同，您可点击【申请修改】更改【银行预留手机号】。

  • 如何进行实名认证？

    1、您需要在民生银行完成开户操作。在开户页面输入【姓名】、【身份证号】、【银行卡号】，在确无误后，点击开通【开启民生银行资金存管】按钮；

    2、跳转到民生开户页面，设置【交易密码】、验证【银行预留手机号】完成后，开户成功。若您【开户手机号】（即在人人贷注册的手机号）与【银行预留手机号】不同，需分别验证对两个手机号接收短信验证码完成验证。

  • 如何查询民生存管账户？

    进入人人贷官网renrendai.com【我的账户】-【账户设置】-【账户信息】中，点击【进入存管账户】即可查看您的民生存管账户信息及交易明细。

  • 实现资金存管后，如何设置交易密码？

    在用户开通民生银行存管账户时，会要求用户设置民生银行交易密码，作为用户充值、提现、出借时的交易密码。若用户需要修改、找回或重置民生银行交易密码，可进入人人贷官网renrendai.com【我的账户】-【账户设置】-【账户信息】中，点击【进入存管账户】，进入民生银行账户管理页面的“密码管理”后，可修改交易密码。

  • 为什么要进行安全认证？

    在升级民生银行资金存管之后，为了保障用户资金的安全性和合同的有效性， 人人贷要求所有出借人必须完成民生银行的身份证绑定、手机号绑定以及交易密码设置，才可以进行出借行为。安全认证的过程简单便捷，民生银行会对于所有个人资料均作严格保密。

  • 进行实名认证后，能解绑或重新认证吗？

    用户在网站不可以进行解绑操作，如需解绑请拨打95770客服热线

  • 一个手机号能绑定几个renrendai.com账户？

    一个。

  • 如何修改绑定手机号？

    用户在人人贷官网renrendai.com打开【我的账户】-【账户设置】-【账户信息】页面，点击修改【绑定手机】后按照提示进行操作，即可修改绑定手机号。

  • 如何更换绑定邮箱？

    打开【我的账户】--【账户设置】--【账户信息】页面，用户可以选择通过原邮箱或绑定手机号修改绑定邮箱。

  • 何时需要进行首笔支付验证？如何完成验证？

    当您在民生银行页面开通存管账户时，若银行预留手机号非您填写的银行卡对应手机号，需要通过首笔支付完成该账户验证。

    具体验证方式参照【首笔支付完成开户】页面，您需要通过您填写的指定银行卡，向您的存管账户转账指定金额（转账金额请查看民生银行短信）。

  • 怎么注销账户？

    符合以下条件的用户，可申请注销您的民生银行存管账户：您已开通民生银行存管账户，账户余额为0，无投标中、退出中的资产，无未结清的借款。在电脑端人人贷官网renrendai.com登录人人贷账户后，进入【我的账户】-【账户设置】-【账户信息】页面，点击【注销账户】后，进入民生银行注销账户页面，按照页面提示进行相关操作即可。

  • 注销账户后，我还能登录人人贷网站或者App继续出借吗？

    您注销民生银行存管账户后，仍可登录人人贷官网或者App，但不能继续出借，如您想继续出借，需重新开通民生银行存管账户。

绑卡

  • 如何完成绑卡操作？

    为了您的资金安全，同一账号同时只能绑定一张银行卡。

    只能绑定您本人（需与民生开户时身份信息一致）持有的银行借记卡，并填写正确的银行预留手机号。

  • 绑卡支持哪些银行？

    支持充值和提现绑卡的银行包括：工商银行、农业银行、中国银行、建设银行、招商银行、交通银行、民生银行、中信银行、兴业银行、光大银行、邮政储蓄银行、广发银行、浦发银行、平安银行、华夏银行。

  • 推荐绑定的银行卡

    为便于您出借，推荐您绑定以下快捷充值限额较高的银行卡：建设银行、交通银行、民生银行、光大银行、兴业银行、浦发银行。

    如绑定邮政储蓄银行卡，进行快捷充值前需在云闪付App开通无卡支付功能。

    如绑定其他银行卡遇快捷充值问题，请使用转账充值方式操作，具体操作可拨打官方客服热线95770，有专业客服团队为您提供帮助。

  • 绑卡成功后可以解绑吗？

    绑卡成功后，如需更换银行卡，可在renrendai.com电脑端-我的账户-P2P银行卡页面,对已绑定的银行进行解绑操作，解绑成功后再绑定其他银行卡即可；或在App（v5.6.1及以上版本）我的账户-我的头像-我的银行卡页面进行解绑。

  • 何时需要进行首笔支付验证完成绑卡？如何验证？

    当您在民生银行页面开通存管账户及绑卡时，若银行校验您的预留手机号填写错误，则民生银行会给您发送一条短信，需要您完成一笔小额转账以验证该账户，完成绑卡操作。

    具体验证方式可参照对应操作流程中的【首笔支付完成开户】页面，您需要使用您开户绑卡时填写的银行卡，向指定账户转账指定金额（转账金额请以民生银行短信为准）。成功转入后，资金可在您的人人贷账户余额中查看。您可使用手机银行、网上银行或前往银行柜台，操作转账充值。

  • 如何申请开通网上银行/手机银行？

    目前所有商业银行基本都支持个人网银/手机银行业务，您可携带有效身份证件，到当地您所持银行卡的发卡行的营业网点，申请开通网上银行/手机银行业务。以上仅供参考，具体业务办理所需材料和办理规则以对应的商业银行规则为准。

  • 是否可以绑定别人的银行卡？

    不可以，只能使用本人的银行卡。

  • 用户在民生银行存管上线前绑定的银行卡是否可以继续使用？

    当存管上线后，用户所有资金划付和交易均将通过用户新开通的存管账户进行，因此用户需要为其新开通的民生银行存管账户进行绑卡操作，并且在绑定之后才可以继续进行充值、提现等操作。

提现

  • 如何申请提现

    登录人人贷财富App或官网后，您可在账户页面随时发起，将民生账户可用余额中的资金申请提现到您所绑定的银行卡中。

  • 提现到账时间及未到账原因

    1、 在您申请提现后，资金预计2个工作日内到账，到账后将有短信提示，请您提前做好资金使用规划并耐心等待。

    2、 若提现金额在5万以上需要在提现时选择正确的开户行。

关于积分

  • 积分是什么？

    积分是人人贷平台推出的一项用户福利，用户可以通过出借或参与平台活动等方式获得积分。所得积分可用于出借抵扣或参与平台活动。

  • 积分如何获得？

    1、 成功开通民生银行存管账户可获得10积分。

    2、 首次完成出借可获得15积分。

    3、 首次完成风险测评可获得10积分。

    4、 首次升级至橙卡及以上等级的会员，可享有相应等级的积分奖励。

    5、 自平台会员体系上线后，普通及以上等级会员发生出借行为，每日即可获得一定数量的积分。每日可得积分=昨日在投资金*0.05*积分倍数/365，其中，昨日在投资金指会员体系上线后用户出借U享、优选、薪享、散标及债权的本金（在自动投标服务中，如果该部分本金产生的利息进行再出借，则再出借的利息部分也计入昨日在投资金）；会员体系上线前用户出借的本金以及再出借的利息（如有）均不计入昨日在投资金，积分倍数根据发放积分时用户所在的会员等级确定。

    6、 邀请好友完成首次出借（不包含散标、债权），邀请人可获得25积分。

    7、 更多积分获取途径以平台实际发布情况为准。

  • 积分如何使用？

    1、 出借时可使用积分进行抵扣，每100积分可抵扣1元人民币。抵扣时使用的积分为100的整数倍，且单笔最高可抵扣出借本金的1%。积分抵扣限于出借或追加优选服务、U享自动投标服务（服务期限≥12个月），U享续期至U享服务（服务期限≥12个月），优选转投至U享服务（服务期限≥12个月），优选转固定期至12个月及以上服务期限；积分抵扣可与优惠券同时使用。

    2、 选择续期至U享，如对续期的U享进行取消/修改续期操作，已选择使用的积分不予返还；如果用户选择修改续期授权出借金额，且在修改前已使用积分抵扣的金额大于修改后授权出借金额的1%，则续期的服务仅按照修改后的授权出借金额的1%进行抵扣，若修改前已使用积分抵扣的金额不足修改后授权出借金额的1%，则按照已使用积分抵扣的金额进行抵扣。抵扣金额在实际进入续期之后发放至您的账户，实际发放抵扣金额以进入续期时的授权出借金额计算为准。

    3、 积分可用于参与转盘抽奖等活动，更多详情可进入福利页面进行查看。

  • 积分发放时间？

    1、 根据在投资金每日计算所得的积分，将于每日23:59:59时清算，并于次日以1的整数倍进行发放。不足1积分的小数部分暂不发放与展示，待该部分积分值（四舍五入保留两位小数）累计至1时进行发放。

    2、 会员等级升级、开通民生银行存管账户、首次完成出借、首次完成风险测评及邀请好友所得积分将实时发放至个人积分账户。

    3、 参与活动获得的积分，具体发放时间以届时公布的活动规则为准。

  • 积分会过期吗？

    当月获得的所有积分，于次年相同月份的最后一天23:59:59过期。例如：用户于2018年1月任意一日中获得的所有积分，将于2019年1月31日23:59:59过期。

  • 特别说明

    1、用户需遵守人人贷平台公布的积分获取及使用相关规则，通过不正当手段（如作弊行为等）获取积分的用户，人人贷平台有权扣除该用户通过其不当行为获得的部分或全部积分。

    2、如您对人人贷平台积分规则有任何疑问，您可致电95770向人人贷平台咨询。网贷有风险，出借需谨慎，出借人风险自担。

  • 现金券使用规则

    现金券可用于抵扣相应面值的出借本金，在出借本金不少于现金券标注的最低出借本金时方可使用。例如：面值100元、最低出借本金20,000元的现金券，您在出借20,000元及以上时才可以使用，使用后将在实际扣除账户余额时减免100元。

  • 抵扣券使用规则

    抵扣券可用于抵扣按抵扣比例所计算的金额的出借本金，即按照券面限定的抵扣比例计算的抵扣金额获得抵扣优惠，抵扣金额=出借本金*抵扣率，且抵扣金额不能超过抵扣券的面值，若计算的抵扣金额大于抵扣券面值，则以抵扣券面值为最终抵扣的出借本金金额。

  • 加息券使用规则

    加息券：用于增加当期U享服务或优选服务的扣费后年利率，加息券分为整个期限加息和短期加息两种。

    1、整个期限加息：使用加息券后，当期U享服务或优选服务将按照增加后的扣费后年利率计算扣费后的利息，追加出借不可使用加息券，当期U享服务续期后不再享受加息。

    2、短期加息：短期加息是由人人贷提供的一种奖励资金，短期加息的加息券会注明加息天数，使用加息券后，当期U享服务或优选服务的扣费后年利率不变，加息券注明的加息天数届满后您将获得一笔奖励加息，奖励加息金额=出借本金*加息部分的扣费后年利率/365*加息天数（使用短期加息券后，当期U享服务或优选服务追加的金额不再适用该优惠券）。

    加息天数届满且您截至加息天数届满日的次日9:00仍享有该期的U享服务或优选服务的，则奖励加息金额将在加息天数届满日的次日9:00后发放至您的账户余额，您可用于提现或出借。

  • 续期加息券使用规则？

    续期加息券：仅在U享服务续期时使用，应于续期所选择的服务的加息，续期加息券分为整个期限加息和短期加息两种。

    1、整个期限加息：可增加续期后U享服务或优选服务的扣费后年利率。

    2、短期加息：短期加息是由人人贷提供的一种奖励资金，短期加息的加息券会注明加息天数，使用续期加息券后，续期后U享服务或优选服务的扣费后年利率不变，续期加息券注明的加息天数届满后您将获得一笔奖励加息，奖励加息金额=出借本金*加息部分的扣费后年利率/365*加息天数，加息天数届满且您截至加息天数届满日的次日9:00仍享有该期的U享服务或优选服务的，则奖励加息金额将在加息天数届满日的次日9:00后发放至您的账户余额，您可用于提现或出借。

  • 预约券使用规则？

    预约券：预约券是一种预约选择一期服务的权利凭证，使用预约券可在U享服务或优选服务预约授权出借阶段预约授权出借。平台会以活动、积分兑换等不特定形式向用户发放预约券。使用预约券预约授权出借后，不可取消预约。

  • 优惠券的其他使用规则？

    • 预约券与现金券、抵扣券、加息券、续期加息券可以同时使用，但每笔支付仅可使用一张现金券/抵扣券/加息券/续期加息券。

    • 优惠券需在注明的有效期内使用，过期作废。

    • 优惠券仅可在您出借券面指定的服务时使用。

    • 每张优惠券仅限使用一次，优惠券在一个服务中使用后，则该张优惠券使用完毕，不可再用于其他服务。
