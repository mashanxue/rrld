  • 我可以出借吗？ 

    年满18周岁，具有完全民事权利能力和民事行为能力，可以在人人贷网站上进行注册、完成实名认证和民生银行的存管账户开户和绑卡，成为用户。您在人人贷网站选择的出借方式所适用的法律法规、监管规定对您的资格有其他要求的，您还应当符合相应要求。  

  • 怎样进行出借？

    • 请您按照以下步骤进行出借：

    • 1. 在renrendai.com平台上进行注册；

    • 2. 进入民生页面完成存管账户开户及绑定银行卡的操作。注：在民生银行开户时候包括：实名认证，设置交易密码、验证手机号；

    • 3. 账户充值；

    • 4. 浏览U享服务、薪享服务、散标列表、债权转让列表，选择自己感兴趣的投资方式；

    • 5. 根据您所选择的出借方式所适用的法律法规和监管规定的要求，按照网站操作提示完成风险承受能力测评；

    • 6. 确认出借，出借成功。

  • 扣费后利息有多少？

    U享服务

    除U享服务-新以外，U享服务主要有6种，服务期限分别为1个月、3个月、6个月、12个月、24个月、36个月，不同服务期限的U享服务扣费后年利率不同，具体以当期U享服务公布的扣费后年利率为准。按扣费后年利率计算的利息不代表对实际获得利息的承诺，您实际获得利息可能与扣费后年利率不一致。

    薪享服务

    薪享服务的扣费后年利率为8%，具体以renrendai.com平台公示及具体的服务协议约定为准。按扣费后年利率计算的利息不代表对实际获得利息的承诺，您实际获得的利息可能与扣费后年利率不一致。

    优选服务

    优选服务扣费后年利率以网站当日发布为准，具体计算方式见服务详情页。

    散标

    年利率区间7%—24%。

    受让债权

    年利率区间7%—24%。

  • 锁定期有多久？

    U享服务

    除U享服务-新以外，U享服务主要有6种，服务期限分别为1个月、3个月、6个月、12个月、24个月、36个月，到期后自动退出。

    薪享服务

    服务期限12个月，到期后自动退出。

    优选服务

    优选服务目前固定服务期为12个月，固定服务期届满转入自由服务期，自由服务期结束后自动退出。

    散标出借

    借款期限3-36个月，持有90天后可提前转让。

    受让债权

    借款期限3-36个月，随时可提前转让。


  • 选择自动投标工具后是否可以提前退出？

    U享服务

    2018年9月27日及之后发布的U享服务不支持提前退出，9月27日之前发布的U享服务用户在服务期限内可随时申请提前退出，但需要按照约定支付提前退出费用，且实际退出时间取决于U享服务下债权实际转让情况。

    散标债权

    投标形成的散标债权，在完成出借90天后可以通过转出债权的方式退出；在债权交易市场受让的债权，可以随时通过转出债权的方式退出。

    薪享服务

    不支持提前退出。    

    优选服务

    2018年9月27日及之后发布的优选服务不支持固定服务期内提前退出，9月27日之前发布的优选服务在12个月固定服务期内，可在renrendai.com网站操作提前退出，但需要按照约定支付提前退出费用； 固定服务期届满转入自由服务期后可在renrendai.com网站和app端随时申请退出，退出免退出费用，退出金额可灵活选择。 

    • 出借是否有额外费用？

      出借过程中可能产生的费用如下：

      1.充值/提现由第三方支付机构、金融机构收取，或者由renrendai.com平台收取以补充renrendai.com平台向第三方支付机构、金融机构支付的充值/提现费用。

      充值费用

      目前用户充值不需要支付充值费用。

       

      提现费用

      单笔提现1万元（含）以内，免收手续费；单笔提现1万元以上，手续费收取2元/笔。

      • 2. U享服务相关费用，详情可参见U享服务费用。

      • 3. 薪享服务相关费用，详情可参见薪享服务费用。

      • 4. 就散标或债权进行债权转让时产生的债权转让服务费（目前为转出价格的0.5%）。

    • 什么是等额本息？

      等额本息还款法是一种被广泛采用的还款方式。在还款期内，每月偿还同等数额的借款(包括本金和利息)。借款人每月还款额中的本金比重逐月递增、利息比重逐月递减。

      具体计算公式如下：

          : 每月还款额

          : 借款本金

           : 月利率

           : 还款总期数

      因计算中存在四舍五入，最后一期还款金额与之前略有不同。

    • 什么是先息后本？

      先息后本是每月支付利息，到期还本的方式。具体计算公式如下：

      P=A·b%

      P：每月还款额

      A：借款本金

      b：月利率

    • 散标怎样计算利息？

      用1万元乘以12%计算是错误的。
