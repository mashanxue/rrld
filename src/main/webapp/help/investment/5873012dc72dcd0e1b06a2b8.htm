  • U享服务利息

    U享服务的扣费后年利率根据具体的服务期限不同存在差异，具体以人人贷理财平台及renrendai.com平台公示及具体的服务协议约定为准。

    人人贷可能不定时调整U享服务扣费后年利率，具体请以平台实际发布的信息为准，按扣费后年利率计算的利息不代表对实际获得利息的承诺。

    相关阅读：利息处理方式     U享服务安全吗？

  • 薪享服务利息

    薪享服务服务期限12个月，扣费后年利率8%，但根据系统自动投标的标的利率不同和借款人的实际履约情况不同，每个出借人的实际获得利息存在少许差异。

  • 散标利率

    年利率区间7%—24%，具体利率由您所出借的借款标的利率确定。

  • 受让债权的利率

    年利率区间7%—24%，具体利率由您所出借的转让债权利率及出让人设定的折让系数确定。

  • 等额本息借款的收益该如何计算？

    • 等额本息还款法是一种被广泛采用的还款方式。在还款期内，每月偿还同等数额的借款(包括本金和利息)。借款人每月还款额中的本金比重逐月递增、利息比重逐月递减。

    具体计算公式如下：


        : 每月还款额

        : 借款本金

         : 月利率

         : 还款总期数

    因计算中存在四舍五入，最后一期还款金额与之前略有不同。

  • 充值提现费用

    充值费用

    目前用户充值不需要支付充值费用。


    提现费用

    单笔提现1万元（含）以内，免收手续费；单笔提现1万元以上，手续费收取2元/笔。

  • U享服务费用

    授权出借U享服务服务后，可能产生的费用包括：

    授权出借费用：

    a%；授权出借服务本金的a%，期初额外收取，即授权出借服务本金为10万元，则另行收取10万元*a% 作为授权出借服务费用归人人贷所有。（当前授权出借费率为0.0%）

    服务费用：

    U享服务对应借款实际利息收入中，超过U享服务扣费后年利率以外的部分作为服务费用归人人贷所有，若实际获得利息不及扣费后年利率的，人人贷不收取服务费用。详情参见《U享服务服务协议》。

    提前退出费用（2018年9月27日及之后发布的U享服务不支持服务期内提前退出，2018年9月27日之前发布的U享服务在服务期内提前退出按以下规则计费）：

    c%；提前退出U享服务时按授权出借服务本金的c%收取，提前退出费用归人人贷所有。(当前提前退出费率的标准为：除服务期限为36个月的U享服务外，提前退出费率为2.0%，服务期限为36个月的U享服务提前退出费率为3%)。

  • 薪享服务费用

    授权出借或退出薪享服务不产生任何费用，人人贷收取回款利息的10%作为服务费。

    但是薪享服务有延期费用，若您当月未能按时支付薪享服务的月出借本金即视为当月发生延期，延期费用=（薪享服务各月累计延期天数-10）/365*薪享服务到期总利息。

    注：薪享服务的扣费后年利率是扣除相关服务费后的值。

  • 散标费用

    散标出借无费用。

  • 债权转让服务费

    债权转让的费用为转让服务费。人人贷向转出人收取，不向购买人收取任何费用。

    转让服务费金额为成交金额*转让服务费率，转让服务费率在目前的运营中按0.5%收取，具体金额以债权转让页面显示为准。债权转让服务费在成交后直接从成交金额中扣除，不成交renrendai.com平台不向用户收取转让服务费。
