法律与政策保障

  • renrendai.com平台提供居间撮合服务的合法性

    《合同法》第23章专门对“居间合同”作出规定，其第424条明确定义为：“居间合同是居间人向委托人报告订立合同的机会或者提供订立合同的媒介服务，委托人支付报酬的合同”。renrendai.com平台是合法设立的网络借贷信息中介平台，致力于为民间借贷业务提供优质高效的撮合服务，以促成借贷双方形成借贷关系，然后收取相关报酬。此种居间服务有着明确的法律依据。

  • 出借人及借款人之间的借贷关系的合法性

    《合同法》第196条规定：“借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同”；根据《合同法》第十二章“借款合同”和《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》，我国法律允许自然人等普通民事主体之间发生借贷关系，并允许出借方到期可以收回本金和符合法律规定的利息。出借人与借款人之间形成的借贷关系受到法律保护。

  • 出借人通过renrendai.com平台获得的利息的合法性

    根据《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》第二十六条：“借贷双方约定的利率未超过年利率24%，投资人请求借款人按照约定的利率支付利息的，人民法院应予支持。借贷双方约定的利率超过年利率36%，超过部分的利息约定无效。借款人请求投资人返还已支付的超过年利率36%部分的利息的，人民法院应予支持。” renrendai.com平台上出借人向借款人出借资金并按照约定利率收取利息，该利率未超过前述规定的上限，为合法利息回报，受到法律保护。

  • 去哪里查询我的出借合同

    服务类可进入官网renrendai.com或App端账户页面，进入持有中服务详情页查询服务协议；散标债权可登录renrendai.com官网-【我的账户】-持有中散标债权列表页点击【合同查询】或在App端【账户】-【债权】，进入持有标的详情页查询出借协议。

  • 电子合同的合法性

    根据《合同法》第11条的规定，当事人可以采用合同书、信件和数据电文（包括电报、电传、传真、电子数据交换和电子邮件）等形式订立合同。电子合同是法律认可的书面合同形式之一。人人贷采取用户网上点击确认的方式签署电子合同。点击确认后的电子合同符合《中华人民共和国合同法》规定的合同成立、生效的要件，其有效性也被人民法院的司法实践所接受。

借款审核与风控

  • 严格的贷前审核

    在客户提出借款申请后，人人贷或人人贷的合作机构会对客户的基本资料进行分析。通过网络、电话及其他可以掌握的有效渠道进行详实、仔细的调查。避免不良客户的欺诈风险。在资料信息核实完成后，根据个人信用风险分析系统进行评估，由经验丰富的借款审核人员进行双重审核确认后最终决定批核结果。

  • 完善的贷后管理

    如果用户逾期未归还借款，人人贷或人人贷合作机构的贷后管理部门将第一时间通过短信、电话等方式提醒用户进行还款。如果用户在5天内还未归还当期借款，人人贷或人人贷的合作机构将会联系该用户的紧急联系人、直系亲属、单位等督促用户尽快还款。如果用户仍未还款，交由专业的高级催收团队与第三方专业机构合作进行包括上门等一系列的催收工作，直至采取法律手段。

资金存管安全升级

  • 什么是资金存管？

    根据《网络借贷信息中介机构业务活动管理暂行办法》第二十八条“网络借贷信息中介机构应当实行自身资金与出借人和借款人资金的隔离管理，并选择符合条件的银行业金融机构作为出借人与借款人的资金存管机构。”和第三十五条“资金存管机构对出借人与借款人开立和使用资金账户进行管理和监督，并根据合同约定，对出借人与借款人的资金进行存管、划付、核算和监督。”“资金存管”是指这样一种安排，即网络借贷信息中介平台在存管银行设立独立账户，将客户资金存入该等独立账户，并由存管银行根据出借人、借款人发出的指令和合同约定，对该等资金的使用、划付进行管理和监督。

  • 实现资金存管对用户有哪些好处及平台的变化是什么？

    首先，资金存管实现了网络借贷信息中介平台自有资金和用户资金的分离，避免网络借贷信息中介平台自有资金与用户资金发生混同，当网络借贷信息中介平台自身发生负债、破产时，用户资金不会受到网络借贷信息中介平台债权人的追索。其次，资金存管是由银行业金融机构根据出借人、借款人的指令和合同约定对用户资金的使用、划付进行管理、监督，且由银行对客户资金交易进行记录，能够尽可能地避免网络借贷信息中介平台非法动用出借人、借款人的资金，保障出借人、借款人的资金安全。

    资金存管上线后，人人贷引入民生银行作为存管银行。每位出借用户都将在民生银行开通独立的个人存管账户。因此在用户出借前增加了民生银行开户操作。在用户进行充值、出借、提现等涉及资金操作时，均需要跳转至民生银行页面校验交易密码，有效保障了用户的资金安全。

    另外由于民生银行对于存管要求，取消了原优选理财服务利息处理方式和退出方式中的提现至银行卡操作。其他业务流程保持不变。

  • 实现民生银行资金存管后，出借人是不是要到民生银行开立账户？

    每位renrendai.com平台的注册用户在进行出借前，均需要在线开通民生银行个人存管账户。无需用户线下前往民生银行进行开户。

  • 银行资金存管具备哪些特征？

    关键点一：用户资金自交易之初就在银行体系内运转, 可实现网络借贷信息中介平台自有资金与用户资金隔离，避免资金池

    将用户资金与renrendai.com平台自有资金进行有效隔离，是renrendai.com平台实现银行资金存管的首要标志。中国民生银行开设专户用于存放renrendai.com平台用户交易资金，从而实现用户资金与renrendai.com平台运营资金的隔离。用户的资金自交易之初就在银行体系内运转。这种隔离避免了renrendai.com平台自建资金池的可能性，消除了renrendai.com平台擅自挪用资金的隐患，从而在根源上保障了用户资金安全。

     

    关键点二：由银行对用户账户进行独立管理

    人人贷与中国民生银行合作的资金存管模式下，民生银行为每一位renrendai.com平台用户设立独立了银行存管子账户。民生银行会对所有用户的交易与资金流水进行簿记，以此实现账户的独立管理，避免平台进行挪用。这改变了过往第三方支付以大账户的方式为平台存管资金和平台自行簿记的做法，使平台用户的资金交易全程都真实在银行端完成。

    同时，民生银行每日将与人人贷就银行系统与平台的总账户，以及所有用户的分账户进行对账，确保数据安全准确。而用户也可在民生银行提供的界面查询资金余额、资产交易情况等数据，避免平台提供不实信息。

     

    关键点三：交易密码由银行管理，平台无法触碰  对用户自主意愿进行安全保障

    在所有涉及资金流转例如充值、出借、提现等环节，renrendai.com平台用户操作时都需要跳转至民生银行页面进行密码验证，确保根据用户真实意愿进行交易。所有用户设置的交易密码由银行系统管理，平台无法触碰。这不仅有效确保每一笔交易的真实性，同时也防止平台在未经用户授权的情况下对用户资金进行违规操作。

  • 用户之前在民生银行存管上线前绑定的银行卡是否可以继续使用？

    当存管上线后，用户所有资金划付和交易均将通过用户新开通的存管账户进行，因此用户需要为其新开通的民生银行存管账户进行绑卡操作，并且在绑定之后才可以继续进行充值、提现等操作。

  • 实现资金存管后，如何完成绑卡操作？

    为了您的资金安全，同一账号同时只能绑定一张银行卡。

    只能绑定您本人（需与民生开户时身份信息一致）持有的银行借记卡，并填写正确的银行预留手机号。

  • 实现资金存管后，如何设置交易密码？

    在用户开通民生银行存管账户时，会要求用户设置民生银行交易密码，此密码将做为用户充值、提现、出借P2P类资产时的交易密码。若用户需要修改或找回民生银行交易密码，也需要进入民生银行管理页面进行操作。

  • 民生银行为我开立的交易资金账户由谁操作，信息安全是否有保障？

    客户的交易结算资金管理账户（即子账户）由客户本人管理，充值、出借、提现时要由客户输入交易密码进行授权，平台或者民生银行都不能自行动用客户账户资金。请客户保护好自己的交易密码、手机验证码等信息，勿向他人泄露。客户可在renrendai.com平台上授权出借U享服务、薪享服务等出借服务，即为开通自动投标功能，自动投标是在客户和renrendai.com平台签订协议后，在客户授权的额度内，由平台根据投标规则为客户发起自动投标。

  • 在renrendai.com平台上注册后，可以不开立银行存管户就进行出借吗？

    不可以。存管子账户用于分账管理客户的交易与资金信息，如不开立，无法进行后续的操作。

  • 如何进入民生银行存管账户？

    点击我的账户-账户设置-存管账户后的文字链接“进入存管账户”来访问民生存管页面，目前只能通过PC端查看账户，无法在APP端查看。

  • 民生交易密码的修改和重置如何操作？

    我的账户-账户设置-账户信息-存管账户-进入存管账户-密码重置。

  • 在验证民生账户/重置密码时，手机收不到验证码怎么办？

    （1）客户手机安装了拦截短信软件；

    （2）客户打电话要求屏蔽民生95568的短信，致使其收不到。需客户电话95568开启短信提醒服务；

    （3）部分运营商短信存在到达率问题；

    （4）目前如果60秒收不到短信，可以点击“语音验证码”，民生系统会通过拨打客户手机号进行语音通知。

  • 我的手机号出现变更，该如何修改？

    客户可以通过平台自行更改。路径为：我的账户-账户设置-账户信息-绑定手机-修改。

  • 是否可以绑定信用卡进行充值或提现？

    不支持，只能使用银行借记卡。

  • 绑卡支持哪些银行？

    支持充值和提现绑卡的银行包括：工商银行、农业银行、中国银行、建设银行、招商银行、交通银行、民生银行、中信银行、兴业银行、光大银行、储蓄银行、广发银行、浦发银行、平安银行、华夏银行。

  • 系统支持的充值渠道有哪些？

    （1）用户在人人贷理财APP端通过充值绑卡进行快捷充值；

    （2）用户在renrendai.com电脑端通过充值绑卡进行快捷充值，或通过网上银行进行网关充值；

    （3）用户通过转账方式（手机银行、网上银行、柜台转账）进行充值，转账时请根据平台提供的收款信息转账，以免造成资金延迟到账。

  • 绑卡成功后是否可以修改？

    如需更换绑定银行卡，可在人人贷官网renrendai.com-【我的账户】-【账户设置】-【我的银行卡】页面，先进行解绑操作；或在App端【账户】-点击左上角【头像】-【银行卡】页面进行解绑操作。解绑成功后，可再重新绑定其他银行卡。

  • 民生银行安全控件安装方法

    如果用户未安装民生银行安全控件，当用户第一次跳转至民生银行页面时，系统会提示用户安装安全控件，下载并按提示安装完成安全控件后，即可正常使用。（chrome浏览器不支持安全控件安装，请使用其他浏览器进行操作）。

  • 为什么我无法下载安装民生银行安全控件？

    MAC系统出现此类问题比较多，WIN10系统不支持下载安装该插件，此外，您的浏览器安装的部分插件会拒绝安装民生银行安全控件。

  • 权限管理和隐私安全

    我们严格遵守国家相关的法律法规，对用户的隐私信息进行保护。未经您的同意， 不会向任何第三方公司、组织和个人披露您的个人信息、账户信息以及交易信息（法律法规另有规定的除外）。

    同时，公司内部也设有严格、完善的权限管理体系，以保证每一位内部员工都只能查看自己职责和权限之内的数据和信息。
